# Waston-D2L

基于《动手学深度学习 Dive into deep learning》的学习整理
原链接：https://zh.d2l.ai/index.html

---
## 项目目标
- 系统性地从零开始掌握深度学习基础
- 用 PyTorch 复现 D2L 中所有核心模型
- 通过自己写代码理解每个模块背后的数学
- 构建一个个人深度学习知识体系
